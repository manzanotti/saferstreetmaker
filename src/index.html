How to use the Safer Street Maker

Welcome to the Safer Street Maker, where you can create the active travel network of your dreams!

It's very early days of the site, so don't be surprised if you encounter the odd issue.

Currently, you can place modal filters (which allow all traffic apart from motor vehicles through), and you can create mobility lanes and tram lines.

Modal Filters

To add a modal filter, click the Green Circle button in the toolbar on the left-hand side of the map, and just click on the map to place one. Once placed, you can click on one to delete it, or drag it to move it.

Mobility Lanes

To add a mobility lane (designed to be used by people on Bikes, Scooters, Wheelchairs and other mobility vehicles), click on the Cycle button in the toolbar. You draw the lane by placing a serious of points. When done, click on the last point added, and you will leave lane creation mode

To edit a lane, click anywhere on it to enter edit mode. You can click on the fainter squares to add an extra point to the line, if you need one. Whilst in edit mode, you will see a popup, which currently has a button to delete the whole lane.

Tram Lines

These work in exactly the same way as mobility lanes. To add one, click on the Tram button in the toolbar. You draw the lane by placing a serious of points. When done, click on the last point added, and you will leave line creation mode.

Car-free Streets

Again, these work in exactly the same way as mobility lanes. To add one, click on the No Vehicles button in the toolbar. You draw the lane by placing a serious of points. When done, click on the last point added, and you will leave line creation mode.

Saving and loading maps

The map is currently automatically saved in the browser you are using. If you wish to either share a map with someone, use it on another device, or have more than one map, click on the Folder icon to open the file menu, and hit the Disk icon to save the current map to a text file on your machine. To load a map, click on the Folder icon to open the file menu, and hit the cloud-like button to select the file you wish to upload.

Exporting maps

If you wish to export the map as GeoJSON, for importing it into a mapping application, click on the Folder icon and hit the GeoJSON icon.

Example map

There is an example map you can download if you want to see what one looks like.

Tech Details

If you're interested, this site is built using the leaflet.js mapping library. It's built using parcel.js. The project is hosted on GitHub. This is a public repository, so if you wish to contribute to the project, feel free! I haven't put a readme together for it yet though. The site is hosted on Azure, as a Static Web App, which is free for a ridiculous amount of traffic.
